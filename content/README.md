# Content

This directory is the public publishing space for Rush Radioo.

- `blogs/`: longform essays, building cards, history cards, and structured articles.
- `feeds/`: concise knowledge notes, concepts, definitions, and observations.
- `_templates/`: starter templates for new entries.

Use `status: "draft"` for private drafts and `status: "published"` for public content.

See `docs/content-workflow.md` for the canonical schemas.
