# 鸱吻
<p style="font-size: small; color: gray;">chīwěn</p>

> 中国古建筑正脊两端的装饰性构件，龙首鱼身，张口吞脊，兼具结构封固与防火寓意。

## 📖 定义与起源

鸱吻（又名**鸱尾、蚩吻、螭吻**）安装于古建筑**正脊两端**，是等级最高的脊饰。其名源于汉代柏梁殿灾后越巫所言："海中有鱼虬，尾似鸱，激浪即降雨"，故以鸱尾形象置于屋脊**厌火避灾**。明代起被纳入"龙生九子"之一，赋予"好望、好吞"的性情。

## 🏛️ 在晋北古建筑中的代表

| 建筑 | 特征 |
|------|------|
| **崇福寺弥陀殿（金）** | 正脊鸱吻高 **3.2m**、宽 **2.95m**，一黄一绿琉璃，张口欲吞，为同期古建**孤例** |
| **大同华严寺大雄宝殿（辽金）** | 鸱吻高 4.5m，八块琉璃拼成，巨口獠牙，龙头鱼尾 |
| **应县木塔（辽）** | 木塔无琉璃鸱吻，塔刹为金属构件，但同属朔州辽金建筑体系 |

崇福寺鸱吻为琉璃术首**武春**团队于皇统六年（1146年）烧制，与殿内垂兽、力士同为**金代琉璃孤品**。

## 🔧 演变脉络

| 时期 | 形态特征 |
|------|---------|
| **汉** | 鸱尾初现，鸟形→鱼虬形，厌火寓意 |
| **唐** | 尾部上翘反卷，出现龙口含脊（如佛光寺东大殿） |
| **宋/辽/金** | 张口力度加大，龙头鱼身定型，《营造法式》规范作法 |
| **元** | 盘绕龙形，琉璃色彩厚重（如永乐宫三清殿） |
| **明清** | 完全龙吻化，雕龙鳞、小龙装饰，琉璃拼装+铁杆加固（如太和殿正吻） |

总体趋势：**鱼尾 → 龙首鱼身 → 完整龙形**，装饰渐趋繁复，等级象征加强。

## 📝 学习笔记

- 鸱吻尺寸存在争议：崇福寺鸱吻一说高 3.5m、宽 3.2m，本卡片采用维基百科数据
- "鸱尾"与"螭吻"并非同物：宋代起螭吻从鸱吻中蜕变，造型逐渐龙化
- 民间等级：帝胄用龙吻，百姓只可用鸱尾；晋商徽商多用鳌头形鸱尾，暗含"独占鳌头"
- 崇福寺鸱吻与垂兽、1.5m 琉璃力士同属"金代五绝"之一的琉璃脊饰系统
