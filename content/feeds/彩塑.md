# 彩塑
<p style="font-size: small; color: gray;">cǎisù</p>

## 一句话定义
在泥塑、木石或陶胎表面施以彩绘的雕塑形式，**塑形与着色并重**。

## 核心特征
- **材质**：以黏土为主，掺入棉絮/麻纤维增强韧性，阴干后入窑或自然固化
- **技法**：捏、塑、堆、贴 → 阴干 → 打底 → 矿物颜料彩绘 → 罩保护层
- **色彩**：朱砂、石青、石绿、金箔等矿物颜料，耐光耐老化
- **风格**：写实与装饰性兼具，重神态传神

## 经典代表

| 名称 | 时代 | 特点 |
|------|------|------|
| 敦煌莫高窟彩塑 | 北魏—元 | 492窟，2000+ 彩塑，飞天与菩萨最具代表性 |
| 麦积山石窟彩塑 | 后秦—宋 | "东方雕塑陈列馆"，泥塑薄肉感极佳 |
| 山西平遥双林寺 | 明 | 2000+ 彩塑，韦驮像被誉为"明代第一塑" |
| 山西晋祠圣母殿侍女像 | 北宋 | 33尊，神态各异，写实巅峰 |

## 与其他雕塑的区别

| 类型 | 区别 |
|------|------|
| vs 石雕 | 彩塑以泥为胎 + 彩绘，石雕以石材本身质感取胜 |
| vs 木雕 | 木雕可有彩绘但胎体不同，彩塑以泥塑为核心 |
| vs 陶瓷 | 陶瓷需高温烧制，彩塑多为低温或阴干 |

## 文化意义
- 佛教东传与中国本土审美融合的产物
- 承载宗教叙事、民俗信仰与时代审美
- 2021年"中国传统彩塑技艺"列入国家级非遗
